About

I'm a game developer passionate about building games from the ground up — from core mechanics and multiplayer networking to UI systems and game feel.

My work lives at the intersection of technical architecture and player experience. I care deeply about how things feel to play, not just how they work under the hood.

Currently focused on Unity multiplayer with server-authoritative architecture using Netcode for GameObjects — building prototypes and learning by shipping.
